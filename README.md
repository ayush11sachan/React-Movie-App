# 🎬 React Movie Search App

## 🚀 Overview

A React-based Movie Search App that fetches real-time data from the OMDb API and displays movies in a card layout.

---

## ✨ Features

* Search movies by name
* Display movie poster, title, and year
* Loading and error handling
* Card-based UI with hover effect

---

## 🛠 Tech Stack

* React (useState)
* JavaScript (ES6+)
* OMDb API
* Vite

---

## 📂 Project Structure

```
src/
 ├── components/
 │    ├── MovieApp.jsx
 │    └── MovieCard.jsx
 ├── App.jsx
 └── main.jsx
```

---

## 🧠 Learning Outcomes

* Understanding React state management using useState
* API integration using fetch()
* Handling asynchronous operations
* Rendering dynamic lists using map()
* Creating reusable components
* Managing loading and error states
* Using environment variables securely
